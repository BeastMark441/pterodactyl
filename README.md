# Pterodactyl Panel

Pterodactyl® — это бесплатная панель управления игровыми серверами с открытым исходным кодом, созданная на PHP, React и Go. С учётом требований безопасности Pterodactyl запускает все игровые серверы в изолированных Docker-контейнерах, предоставляя конечным пользователям красивый и интуитивно понятный интерфейс.

Хватит соглашаться на меньшее. Сделайте игровые серверы полноценной частью вашей платформы.

### Поддерживаемые игры

Pterodactyl поддерживает широкий набор игр, используя Docker-контейнеры для изоляции каждого экземпляра. Это даёт возможность запускать игровые серверы без перегрузки машин множеством дополнительных зависимостей.

Некоторые из основных поддерживаемых игр:

* Minecraft — включая Paper, Sponge, Bungeecord, Waterfall и другие
* Rust
* Terraria
* Teamspeak
* Mumble
* Team Fortress 2
* Counter Strike: Global Offensive
* Garry's Mod
* ARK: Survival Evolved

Помимо стандартного набора поддерживаемых игр, сообщество постоянно расширяет возможности этого ПО, и доступно ещё множество игр, предоставленных сообществом. Среди них:

* Factorio
* San Andreas: MP
* Pocketmine MP
* Squad
* Xonotic
* Starmade
* Discord ATLBot и большинство других Node.js/Python Discord-ботов
* [и многие другие...](https://pterodactyleggs.com)
